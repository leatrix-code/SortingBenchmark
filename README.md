# 🔄 Экспериментальное исследование и сравнение эффективности алгоритмов сортировки пузырьком и слиянием на платформе .NET 10.0

---

## 📋 Описание проекта

Этот проект представляет собой комплексное экспериментальное исследование производительности двух популярных алгоритмов сортировки:

- **Bubble Sort** — простой алгоритм сортировки обменами с временной сложностью O(n²)
- **Merge Sort** — эффективный алгоритм «разделяй и властвуй» с временной сложностью O(n log n)

Проект реализует научное тестирование с использованием репрезентативной выборки данных, высокоточным измерением времени выполнения и подсчётом операций для валидации теоретических оценок сложности.

---

## 🎯 Ключевые возможности

✅ **Реализация на C# 14.0** с современным синтаксисом  
✅ **Подсчёт операций** — точный счётчик сравнений и обменов  
✅ **Высокоточное измерение** — `System.Diagnostics.Stopwatch` для микросекундной точности  
✅ **Множественные типы данных** — случайные, отсортированные, обратные, почти отсортированные  
✅ **Репрезентативная выборка** — 100-1000 уникальных тестовых наборов на размер

## 🔬 Методология тестирования

### Репрезентативная выборка

- Для каждого размера массива генерируется **100–1000 уникальных наборов данных**
- Все наборы имеют **идентичную размерность**
- Используется **фиксированный seed (42)** для воспроизводимости

### Изоляция измерений

Время выполнения измеряется **исключительно для рабочих функций**

**Исключаются операции**:
- Генерация тестовых данных
- Выделение/освобождение памяти
- Инициализация переменных
- Вывод результатов

### Подсчёт операций

Для каждого алгоритма отслеживаются именованные операции:

| Алгоритм | Операции |
|---|---|
| **Bubble Sort** | Сравнения + Обмены |
| **Merge Sort** | Сравнения |

### Time Complexity (Временная сложность)

```
┌──────────────────┬─────────────┬──────────────┬──────────────┐
│   Алгоритм       │   Best      │    Average   │    Worst     │
├──────────────────┼─────────────┼──────────────┼──────────────┤
│ Bubble Sort      │   O(n)      │    O(n²)     │    O(n²)     │

### Space Complexity (Пространственная сложность)

```
┌──────────────────┬──────────────┐

├──────────────────┼──────────────┤

```
Bubble Sort: ✅ Стабильна   (сохраняет порядок равных элементов)

```

## 🛠️ Технические детали реализации

### Оптимизация Bubble Sort

```csharp
// ✨ Ключевая оптимизация: флаг swapped
if (!swapped)
    break;  // ← Выход при O(n) на отсортированных данных
```

**Эффект**: Лучший случай улучшается с O(n²) до O(n)

### Стабильность Merge Sort

```csharp
// ✨ Нестрогое неравенство обеспечивает стабильность
if (leftArray[i] <= rightArray[j])  // ← <= вместо <
    arr[k++] = leftArray[i++];
```

**Эффект**: Элементы с равными ключами остаются в исходном порядке

### Предотвращение переполнения

```csharp
// ✨ Безопасное вычисление середины
int mid = left + (right - left) / 2;  // ← Вместо (left + right) / 2
```

**Эффект**: Избегание целочисленного переполнения на больших индексах
